HTML

O que é?

O HTML é uma linguagem de programação de HiperTexto, logo ele não é sequencial.

Para que serve?

Ele é utilizado na criação de sites junto com o CSS, possuindo tags, classes, id que ajudam na marcação de elementos.

O que são Tags?

Basicamente, as tags são elementos que irão definir a estrutura de um site. Elas servem para organizar os conteúdos presentes na página web.

Exemplos de Tags:

  • h1: titulo
  • p: parágrafo
  • img: imagem
  • table: tabela

    • CSS

    O que é?

    O CSS é uma linguagem de estilização, a sua tradução seria "Folhas de Estilo em Cascata", isso se deve as regras de estilo que são aplicadas nas páginas web
    de forma hierárquica, ou seja, em cascata.

    Para que serve?

    É uma linguagem utilizada para estilizar e formatar sites, tornando assim, as páginas web mais atraentes. Ele também trabalha muito bem junto ao HTML para definir
    a aparência visual, mudando as cores, fontes, layouts, espaçamento etc.

    Exemplos de propriedades:

  • color: cor do texto
  • text-align: alinha o texto
  • font-family: define a fonte
  • font-size: define o tamanho da fonte
  • margin: define a margem ao redor do elemento

    • Git/GitHub

    O que são?

    A diferença entre os dois é que o Git é um programa de código aberto e o GitHub é uma plataforma de Hospedagem de código-fonte.

    Para que servem?

    O Git serve para o controle de versões de projetos software. O GitHub, como já mencionado anteriormente, serve para hospedar o código-fonte, sendo muito útil em projetos,
    pois nele é possível criar repositórios, facilitando na colaboração entre usuários.


    JavaScript

    O que é?

    É uma linguagem de programção principalmente orientada a objetos e funcional.

    Para que serve?

    Permite adicionar interatividade às páginas web, sendo executado no navegador do usuário, vale ressaltar que é uma linguagem muito versatil, visto que é utilizado no
    desenvolvimento web, aplicativos móveis e até jogos. Dentro do JavaScript é possível criar e manipular arrays, funções, variáveis etc.

    Exemplos de funções e utilizações:

  • Math.random(): retorna um número aleatório dentro das predefinições do usuário.
  • Math.ceil(): Arredonda um número para o inteiro mais próximo acima.
  • .toUpperCase(): Converte todos caracteres para maiúsculo dentro de uma string.
  • .split(): Divide uma string em um array.
  • .push(): Adiciona um ou mais elementos no final de um array.
